## Documentation Structure

### 기획 문서
- **PRD**: 비전, 목표, 기능 명세
- **UI/UX**: 디자인 시스템, 와이어프레임

- **아키텍처**: 시스템 구조, 데이터 흐름

### 운영 문서
- **배포**: CI/CD, 환경 설정
- **마케팅**: 릴리스 계획, 홍보 전략
