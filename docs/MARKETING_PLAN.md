## 1. 마케팅 개요

### 1.1 브랜드 포지셔닝

**브랜드 슬로건**:
```
"WaterTime - 당신의 수분 코치"
```

**핵심 메시지**:
```
건강한 수분 섭취 습관을 만드는 가장 쉬운 방법
```

**브랜드 퍼스널리티**:
- 친근한 (Friendly)
- 믿을 수 있는 (Trustworthy)
- 동기부여하는 (Motivating)
- 과학적인 (Scientific)

### 1.2 타겟 오디언스

**Primary Target**:
```
나이: 20-40세
성별: 남녀 모두
특징:
- 건강 관심 높음
- 스마트폰 적극 활용
- 바쁜 일상 속 건강 관리 필요
- 앱 사용에 익숙함
```

**Secondary Target**:
```
나이: 40-60세
특징:
- 건강 관리 필수적
- 만성 질환 예방 관심

```
