## 3. 사용자 플로우

### 3.1 첫 실행 플로우

```
앱 실행
  ↓
스플래시 (2초)
  ↓
온보딩 (3페이지 스와이프)
  ↓
회원가입/로그인 선택
  ├→ 회원가입 → 프로필 설정 → 홈
  └→ 로그인 → 홈
```

### 3.2 일일 사용 플로우

```
알림 수신
  ↓
앱 열기
  ↓
홈 화면 (현재 상태 확인)
  ↓
[+ 버튼] 또는 [위젯]
  ↓
빠른 기록 모달
  ↓
용량 선택 (1탭)
  ↓
기록 완료 (애니메이션)
  ↓
홈 화면 (업데이트된 프로그레스)
```

### 3.3 통계 확인 플로우

```
하단 탭 [통계] 클릭
  ↓
통계 화면
  ↓
[일] [주] [월] 탭 전환
  ↓
그래프 및 인사이트 확인
  ↓
[PDF 리포트] (프리미엄)
```

### 3.4 설정 변경 플로우

```
하단 탭 [설정] 클릭
  ↓
설정 화면
  ↓
원하는 섹션 선택
  ├→ 알림 설정 변경
  ├→ 목표 설정 변경
  ├→ 프로필 수정
  └→ 테마 변경
  ↓
변경 사항 저장
  ↓
설정 화면 (업데이트됨)
```

## 4. 인터랙션 디자인

### 4.1 마이크로 인터랙션

#### 물 섭취 기록 시
```
1. 용량 버튼 클릭
2. 버튼 스케일 업 (1.1x, 100ms)
3. 물방울 아이콘이 위로 떠오름 (300ms)
4. 프로그레스 바 즉시 업데이트 (200ms ease-out)
5. "기록 완료!" 토스트 (1.5초)
6. 진동 피드백 (선택사항)
```

#### 목표 달성 시
```
1. 프로그레스 바 100% 도달
2. 색상 변경 (파랑 → 초록)
3. 축하 confetti 애니메이션 (2초)
4. "🎉 오늘 목표 달성!" 모달
5. "SNS에 공유" 버튼
6. 배지 획득 (있는 경우)
```

#### 알림 수신 시
```
1. 상단에서 알림 배너 내려옴
2. "💧 물 마실 시간이에요!"
3. [마셨어요] [나중에] 버튼
4. 마셨어요 클릭 → 바로 기록됨
5. 나중에 클릭 → 15분 후 다시 알림
```

### 4.2 트랜지션

#### 화면 전환
```
- Push: 오른쪽에서 왼쪽으로 슬라이드 (300ms)
- Pop: 왼쪽에서 오른쪽으로 슬라이드 (300ms)
- Modal: 아래에서 위로 슬라이드 (250ms)
```

#### 탭 전환
```
- 페이드 전환 (150ms)
- 콘텐츠는 즉시 전환
```

### 4.3 제스처

#### 스와이프
```
- 온보딩: 좌우 스와이프로 페이지 전환
- 기록 리스트: 위아래 스와이프로 스크롤
- 삭제: 왼쪽으로 스와이프 → 삭제 버튼
```

#### 롱 프레스
```
- 기록 항목 길게 누르기 → 수정/삭제 메뉴
- + 버튼 길게 누르기 → 직접 입력 모드
```
